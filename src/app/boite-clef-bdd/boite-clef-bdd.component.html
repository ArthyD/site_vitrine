Projet site de réservation de vélo

Le Bureau du Développement Durable de IMT Atlantique met à disposition des vélos aux étudiants.

Pour l'instant, la remise des clefs d'antivols de vélos se fait en main propre. Cela pose plusieurs problèmes :

  • Il faut une personne toujours présente pour donner des clefs
  • Il est difficile d'avoir un historique des réservations (surtout en cas de problème)
  • La récupération des clefs est compliqué

C'est pourquoi l'association a décidé de créer une boit à clef autonome permettant aux étudiants de réserver en ligne leur vélo et le récupérer simplement quand ils le souhaitent.

J'ai donc travaillé sur ce projet dont le résultat peut être retrouvé sur mon github : ici.

Le site web

Le site web a été réalisé à l'aide de python Flask.

L'API permet de gérer une base de données d'utilisateurs, de réservations et de vélos à l'aide de SQLite par soucis de simplicité.

Afin de rendre la page web intuitive et jolie, le framework bootstrap a été utilisé.

La partie arduino

Le serveur web communique avec la boîte physique. Cette boîte utilise un arduino MKR1010, connecté au même réseau que le serveur.

Ainsi le serveur peut donner l'ordre à la boîte de s'ouvrir quand un utilisateur veut emprunter ou rendre un vélo.
